# Htadmin V0.1 后台cms框架

### 介绍
htadmin是基于Thinkphp5.0.24，算是tp5.0.x的最新版的后台管理系统，基本上融入常用的cms简单功能。

### 软件架构
htadmin是基于Thinkphp5.0.24，更新为最新的，但是好像最新版本也比较久了，最新6.x都出来了，喜欢这个版本的可以用。
htadmin是一个通用的后台管理系统，包含功能不多，有需要的下载使用哈，有bug也请反馈。
